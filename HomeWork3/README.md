#### 3. Оценка атрибутов качества 
1. Производительность
Так как все сервисы взаимодействуют синхронно, можно сказать, что процесс создания заявки должен быть быстрым
2. Масштабируемость
В случае необходимости и при правильной реализации всех сервисов, можно гибко горизонтально масштабировать ту часть, которая этого потребует
3. Надежность
В случае отказа одной из частей, процесс подачи заявки будет невозможен
4. Модифицируемость
Основным оркестратором является Web App, его модифицировать будет сложнее всего. Остальные части отвечающие за одну задачу можно легко модифицировать
